### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 预览旅游规划(购买后查看完整内容)
4. 发布旅游规划
5. 评论区
6. 购物车
7. 订单记录
8. 安全退出


**管理员功能：**
1. 登录
